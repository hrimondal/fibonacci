### Welcome
Its a simple fibonacci counter.Run this code using python.

**Requirements**

```
Python 3
```

**How to run**

**First**,clone this repo using:
```
git clone https://www.github.com/hrimondal/fibonacci
```

Then,cd to the directory where you stored it.

```
cd (Your directory)
```

Then run,

```py
python fibonacci.py
```
